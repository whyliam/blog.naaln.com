## 📚 阅读

### 从 NotebookLM 看 AI 工具的设计模式

设计负责人杰森·斯皮尔曼（Jason Spielman）对他领导谷歌 AI 产品 NotebookLM 设计的全面回顾。他详细阐述了自己如何将一个早期实验项目，从零开始打造成一款荣获《时代》杂志「2024 年度最佳发明」的成熟产品。他在其中负责了核心用户体验（UX）、品牌标识和视觉系统的完整设计。

#### **核心问题与设计架构**

项目的初衷是为了解决用户在研究和整合信息时普遍面临的「标签页过载」（tab overwhelm）问题，即不得不在多个工具和窗口间频繁切换，导致思路中断。NotebookLM 的目标是创建一个能将阅读、写作和创作过程无缝集成的统一空间。

为实现这一目标，斯皮尔曼设计了标志性的**三栏式结构**。这个看似简洁的布局经过了上千次迭代，最终确立了一个清晰且可扩展的心智模型：**输入 (Inputs) → 对话 (Chat) → 输出 (Outputs)**。

1. **来源面板 (Source Panel - 输入):** 这是用户旅程的起点。用户在此处上传和管理他们的所有原始资料，如文档、笔记、网页链接等。
2. **对话面板 (Chat Panel - 对话):** 这是体验的核心。用户通过与 AI 进行自然语言对话，来查询、分析、总结和综合来源面板中的信息。AI 的回答会附带来源引文，确保信息可靠。
3. **工作室面板 (Studio Panel - 输出):** 这是创作和成果转化的空间。用户可以将对话中产生的洞察和 AI 生成的内容，整理成结构化的笔记、学习指南、专业报告，甚至是创新的「音频概览」。

这三个面板是动态且响应式的，可以根据用户的当前任务灵活调整大小和布局，例如优化为「阅读 + 对话」或「对话 + 写作」等不同模式，确保在任何屏幕尺寸下都能高效利用空间。

#### **设计理念与核心启示**

斯皮尔曼分享了从这个项目中获得的重要经验：

1. **与用户共建，而非仅为用户设计：** NotebookLM 团队规模小而精干，在不到两个月的时间里就将一个想法从原型推向了市场。他们奉行「尽早发布，持续迭代」的原则，通过 Discord 等社区与用户紧密互动，并根据用户的迫切需求（如快速增加了行内引文功能）迅速调整开发方向。
2. **打造 AI 原生体验：** 他强调 AI 功能应该是「内置的」（built-in），而非「嫁接的」（bolted-on）。AI 不应是附加功能，而应是产品核心体验的有机组成部分，真正解决人类面临的问题。
3. **未来的 UI 将更动态化和情境感知化：** AI 的强大能力要求 UI 设计突破传统范式。未来的界面需要能根据用户的任务和意图动态调整，并主动预测需求、推荐相关操作，以降低用户的认知负荷。
4. **「对话」是过渡时期的关键：** 目前，对话式交互（Chat）仍然是用户理解和使用 AI 最自然、最熟悉的方式，它在传统 UI 和未来 AI 优先体验之间扮演着重要的桥梁角色。

原文：[Jason Spielman • Designing NotebookLM](https://jasonspielman.com/notebooklm)

## 🤖 AI

### 生成新世界的系统

OpenAI 研究员姚顺雨认为，AI 的发展已进入「下半场」，其主线程正从模型训练和方法设计，转向如何定义有价值的任务、构建有效的环境以及创造全新的交互方式。他强调，语言是人类为实现「泛化」而发明的最重要工具，因此基于语言的智能体（Language Agent）是通往通用人工智能（AGI）的核心路径。

**Agent 的演进与本质：从「专用」到「通用」的飞跃**

姚顺雨将 Agent 的发展划分为三个时代：

1. **符号主义 AI**：基于规则，但难以覆盖复杂现实，很快遇到瓶颈。
2. **深度强化学习**：在虚拟环境中取得巨大成功（如 AlphaGo），但其能力局限于特定任务，无法泛化。
3. **大语言模型驱动的 Agent**：这是当前的第三波浪潮。其与传统 Agent 的本质区别在于，大模型赋予了 Agent 强大的「推理」能力，使其能够在不同任务和环境间实现「泛化」。一个例证是，通过强化学习训练模型的数学和编程能力，竟能同时提升其创意写作水平，这是前所未有的。
		
**未来的机遇与挑战：记忆、奖励与生态**

展望未来，姚顺雨指出了 Agent 发展的三个核心研究方向：

1. **长期记忆（Memory）**：他认为当前模型最大的瓶颈不是推理能力，而是缺少获取、管理并利用上下文（Context）的能力。解决记忆问题是释放 AI 巨大应用价值的关键。
2. **内生奖励（Intrinsic Reward）**：要让 Agent 实现从「执行者」到「创新者」的跨越，就需要像人类科学家或艺术家一样，拥有内在的好奇心、价值观和自我激励机制，从而在没有外部反馈的情况下也能持续探索。
3. **多智能体（Multi-Agent）**：构建能高效协作的智能体组织，是实现更复杂、宏大任务的基础。

## 🛠️ 工具

### News Minimalist——AI 筛选重要新闻，过滤无关内容

![News Minimalist——AI 筛选重要新闻，过滤无关内容](https://pics.naaln.com/910shots_so.png-basicBlog)

利用 AI 每天分析近 30,000 篇文章，按重要性评分，从而筛选出值得阅读的新闻。用户可以通过每周 2-3 封邮件获取最重要的新闻摘要，或创建个性化的新闻提要。这个平台帮助人们摆脱无关紧要的新闻噪音，只关注世界重大事件
